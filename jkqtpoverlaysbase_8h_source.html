35 /*! \brief this virtual base class describes an interface for graph overlay elements, which are simple geometric
36  forms drawn ONTO the graphe, so a redraw of the overlays does NOT require a redraw of the graph.
37  \ingroup jkqtplotter_overlays
38 
39  These simple primitive elements can be used to e.g. display fast changing indicators on the graph ...
40  */
